# 🌸 Ferns N Petals — Sales Performance Analysis

## 📌 Project Overview

An end-to-end sales analysis for **Ferns N Petals (FNP)**, one of India's leading gifting and floral brands. This project examines sales performance across occasions, product categories, cities, and time periods to identify revenue drivers and support data-driven business decisions.

---

## ❓ Business Problem

The business needed clarity on:
- Which occasions and product categories drive the most revenue
- Which cities and regions are top performers
- How sales fluctuate across months and time of day
- What the average customer spend looks like

---

## 🛠 Tools Used

- **Microsoft Excel** — data cleaning, analysis, dashboard development
- **Power Query** — data transformation
- **Pivot Tables & Charts** — aggregations and visualisations
- **Slicers** — interactive filtering by occasion, category, and time
